\# Effexe New Era • Core (Version B)





\## Installation

1\. Installer Node.js 18+.

2\. Copier `config.example.json` en `config.json` et remplir.

3\. `npm install`

4\. `node index.js` ou utiliser un process manager (pm2).





\## Hébergement conseillé

\- Render / Railway / Replit (ou VPS pour plus de fiabilité)





\## Permissions du bot (intentions)

\- GUILD\_MEMBERS, GUILD\_MESSAGES, MESSAGE\_CONTENT (si nécessaire pour certains logs), GUILD\_BANS
